# Financial Tech Lab

## Estrutura
Microsserviço Estrutura responsável por apresentar a estrutura inicial que será utilizada como base para
a criação dos novos serviços do backend.

### Utilização
Para melhor utilização e copy-paste do código, o ideal é clonar o repositório e copiar os arquivos para o novo serviço.

## Orientação
Utilize a pasta `docs/` para entender o papel de cada arquivo e fazer um onboarding eficiente. 
Os documentos estão divididos por área de responsabilidade, começando pela visão geral da arquitetura, stack, árvore de diretórios e fluxo de execução, e depois aprofundando no código-fonte, build, infraestrutura e guidelines de engenharia.
